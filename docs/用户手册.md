# 用户手册

autodo-engine 提供运行时初始化、图注册、任务推进、审计查询与事务直调能力。

支持两种模式：

- 独立模式：仅安装 autodo-engine，使用引擎内置事务与内置工具。
- 增强模式：同时安装 autodo-kit，自动扩展官方事务与工具。

## 1. 安装

```powershell
uv pip install -e .
```

可选安装增强包：

```powershell
uv pip install -e ../autodo-kit
```

## 2. 初始化运行时

```powershell
python -m autodoengine.main init-runtime --base-dir ./.tmp/runtime_v4
```

## 3. 注册流程图并创建任务

```powershell
python -m autodoengine.main register-graph --graph-file workflows/workflow_20260308_all_affairs_demo/workflow.json
python -m autodoengine.main create-task --title "示例任务" --goal-text "验证运行链路" --current-node-uid node-start
```

## 4. 推进任务

```powershell
python -m autodoengine.main run-task-step --task-uid <task_uid> --graph-uid <graph_uid>
python -m autodoengine.main run-task --task-uid <task_uid> --graph-uid <graph_uid> --max-steps 50
```

## 5. 查看审计与数据库路径

```powershell
python -m autodoengine.main show-task --task-uid <task_uid>
python -m autodoengine.main show-decisions --task-uid <task_uid>
python -m autodoengine.main show-runtime-events --task-uid <task_uid>
python -m autodoengine.main show-runtime-store-paths --base-dir ./.tmp/runtime_v4
python -m autodoengine.main show-affair-registry-paths --workspace-root .
```

## 6. 统一 capability 调用

列出 capability：

```powershell
python -m autodoengine.main list-capabilities
```

执行 capability：

```powershell
python -m autodoengine.main invoke-capability --capability-id affair_refresh --payload-json '{"workspace_root":"."}'
```

校验 capability manifest/schema：

```powershell
python -m autodoengine.main lint-capabilities
```

## 7. Python 方式调用 capability

```python
from pathlib import Path
from autodoengine import api

result = api.invoke_capability(
    "affair_show_paths",
    payload={"workspace_root": str(Path.cwd())},
    workspace_root=Path.cwd(),
)
print(result)
```

## 8. 直接调用事务

```python
from pathlib import Path
from autodoengine import api

api.run_affair(
    "图节点_start",
    config={"output_dir": "output/demo"},
    workspace_root=Path.cwd(),
)
```

复合事务示例：

```python
api.run_affair(
    "引擎能力巡检",
    config={"output_dir": "output/inspection"},
    workspace_root=Path.cwd(),
)
```

## 9. Runner 参数文件调用

```powershell
python scripts/invoke_capability.py path/to/params.json
```

说明：

- 事务解析、冲突检测、用户覆盖规则在本仓执行。
- 已安装 autodo-kit 时，官方事务源码与事务文档优先来自 autodo-kit。
- 未安装 autodo-kit 时，自动回退到 `autodoengine/affairs` 与 `autodoengine.tools`。
- capability 调用会自动产出审计文件到 `.autodoengine/tool_audits/`。
