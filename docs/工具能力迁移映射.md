# 工具能力迁移映射

## 说明

本文档记录 autodo-engine 首批工具能力从根导出语义到 `atomic/public` 平台的迁移映射。

## 映射表

| 旧导出/语义 | 新 atomic 模块 | 新 capability |
| --- | --- | --- |
| show_runtime_store_paths | autodoengine.tools.atomic.path.runtime_paths | runtime_show_paths |
| show_affair_registry_paths | autodoengine.tools.atomic.path.runtime_paths | affair_show_paths |
| refresh_affair_registry_minimal | autodoengine.tools.atomic.docs.affair_registry_tools | affair_refresh |
| list_runtime_affairs_summary | autodoengine.tools.atomic.docs.affair_registry_tools | affair_list |
| check_affair_conflicts_summary | autodoengine.tools.atomic.docs.affair_registry_tools | affair_check_conflicts |
| bootstrap_runtime_tool | autodoengine.tools.atomic.runtime.runtime_tools | runtime_bootstrap |
| load_graph_summary | autodoengine.tools.atomic.graph.graph_tools | graph_load_summary |
| prepare_affair_config_tool | autodoengine.tools.atomic.config.affair_config_tools | affair_prepare_config |
| run_affair_tool | autodoengine.tools.atomic.affair.runner_tools | affair_run |
| import_user_affair_tool | autodoengine.tools.atomic.affair.runner_tools | affair_import_user |

## 兼容策略

- 根模块 `autodoengine.tools` 继续保留旧函数导出。
- 新增 `list_capabilities`、`invoke_capability`、`lint_public_manifest` 作为统一工具平台入口。
- 新增 CLI/Runner/Agent adapter，避免入口层直接访问 atomic 模块。
